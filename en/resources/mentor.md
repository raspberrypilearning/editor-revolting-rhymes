# Mentor notes

## Goal
Creators code a short poem about revolting things. They practise using variables, f-strings, formatting `print()`, string methods, `.replace()`, a simple `for` loop, and basic randomness with `random.choice()`.

## Common issues
- Missing commas in multi-line `print()`.
- Forgetting `sep='\n'` so lines stay squashed.
- Putting `.lower()` outside the `{}` of an f-string.
- Forgetting to add the emoji to the first line.
- Not importing `random` before calling `random.choice()`.
